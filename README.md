To‑Do List App

A simple and user‑friendly To‑Do List web application built using HTML, CSS, and JavaScript.  
This app allows users to add tasks, delete tasks, and manage their daily activities easily.

Features
• Add new tasks  
• Delete tasks  
• Clean and responsive UI  
• Stores tasks temporarily while the page is open  
• Simple and beginner‑friendly code  

How It Works
Type a task in the input box  
Click the “Add” button  
The task appears in the list  
Click “Delete” to remove a task  

Technologies Used
• HTML  
• CSS  
• JavaScript  

Project Structure
``
index.html   → Main HTML structure  
style.css    → Styling and layout  
script.js    → App functionality  
``
